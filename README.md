# Official Website for Student Leader Board  
**SMK Munshi Ibrahim Labis (SMKMIL)**

This repository contains the official website of the **Student Leaders Board (SLB)** of Sekolah Menengah Kebangsaan Munshi Ibrahim Labis. The site serves as a central platform for student leadership information, school announcements, and internal SLB communication.

> **Note:** This project is created exclusively for school purposes and information dissemination within SMKMIL.
